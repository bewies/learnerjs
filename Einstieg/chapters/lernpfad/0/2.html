Variablen
Variablen sind sehr praktische Dinge in einer Programmiersprache. Sie erlauben uns Werte zwischenzuspeichern, um sie später weiter zu verwenden. Dadurch werden manche Dinge erst möglich, andere deutlich einfacher und übersichtlicher.

Syntax
Eine Variable lässt sich verschieden definieren und kann verschiedene Datentypen halten. In JS muss man nicht den Datentyp deklarieren, da sich Variablen an den Datentyp anpassen.
var, let, const
Die Schlüsselwörter var und let sind so ziemlich die gleichen. Sie erzeugen eine Variable, die völlig neu definiert werden oder verändert werden kann. Das Schlüsselwort const hingegen erzeugt eine Konstante. Sie zeichnet aus, dass man ihren Wert, sobald einmal definiert, nicht mehr verändern kann. Auf diese Weise kann man das Rumpfuschen am Code von Nutzern vermeiden.
var a = 5; let b = 10, c = 23; const d = c - (a+b); console.log(d); // Gibt 8 aus d = a + b + c; // Gibt einen Fehler, weil d eine Konstante ist. a = b + c + d; // Gibt keinen Fehler, weil a eine Variable ist. console.log(a); // Gibt 41 aus
Man kann mehrere Variablen oder Konstanten mit einem Statement definieren, indem man die Variablen mit einem Komma trennt. Wie das genau aussieht, siehst du in Zeile 2, wo b und c gleichzeitig definiert sind. Du kannst auch Variablen erstellen, um ihnen eine Kapselung (Scope) zu geben. Beispiele
Wie oben schon erwähnt, können wir mit Variablen den Prozess vom Auswählen von Elementen hübscher und übersichtlicher und sogar ein bisschen schneller machen. Wir werden Übungen machen, die wir schon gemacht haben, aber auch Neue.
EINS, ZWEI, DREI, VIER
Greifen wir das Beispiel der Count-Elemente des Selektorkapitels auf.
<html lang="en"> <head> <title>My First Class-Selector</title> </head> <body> <div class="myCount"></div> <div class="myCount"></div> <div class="myCount"></div> <div class="myCount"></div> <!-- Hier kommt der Skript rein, der die div.myCount manipuliert. --> <script> ... </script> </body> </html>
Durch die Verwendung von einer Variable können wir es folgerndermassen aufbauen:
let countElements = document.querySelectorAll(".myCount"); countElements[0].innerHTML = "EINS"; countElements[1].innerHTML = "ZWEI"; countElements[2].innerHTML = "DREI"; countElements[3].innerHTML = "VIER";
Kurze Erklärung:
Speichere die NodeList aller div.myCount unter dem Namen countElements ab. Dann verändere das erste Element aus der NodeList... et cetera.

Der Vorteil an dieser Schreibweise ist, dass die Abfrage nach der Collection nur ein Mal von Statten läuft und dann gespeichert wird anstatt 4 Mal das selbe abzufragen. Das erhöht die Effizienz. Dadurch, dass man die Variable passend bennenen kann, wird der Code auch lesbarer und leichter zu warten.
Toggle-Flipflop
Ein Toggle-Flipflop ist eine Schaltung, die mit nur einem Eingang bei Aktivierung immer den Zustand wechselt: An oder Aus. Der Sinn der Sache ist, dass gespeichert werden muss, was der Zustand ist, um herauszufinden, wie der Zustand sein wird.
Ist er an (Zustand) mache aus (Änderung).
Ist er aus (Zustand) mache an (Änderung).
<html> <head> <title>My First Toggle-Flipflop</title> </head> <body> <div id="tff">OFF</div> <!-- Hier kommt der Skript rein, der das TFF funktionierend macht. --> <script> ... </script> </body> </html>
Unser Flipflop besteht darin, das innerHTML im TFF-Element zwischen "Aus" und "An" hin und her zu schalten. Für unser folgendes Programm brauchen wir ein paar mehr Dinge, als nur Selektoren und Variablen. Jetzt kommen if-Statements und Funktionen ins Spiel. Sie werden nach der Erarbeitung näher beschrieben sein. Aufgepasst: Wir benutzen den querySelector(), der keine Collection, sondern nur ein HTML-Element ausgibt.
let myTff = document.querySelector("#tff"); let tffOn = false; function toggleFlipFlop(){ if(tffOn){ myTff.innerHTML = "OFF"; }else{ myTff.innerHTML = "ON"; } tffOn = !tffON; }
Kurze Erklärung in Worten:
Wenn das tff an ist (tffOn == true oder einfach nur tffOn), dann setze das innerHTML des tffs auf aus. Ansonsten (else, also wenn tffOn == false oder einfach nur !tffOn) mache es an.
Das letze Statement sagt: Setze die Variable tffON auf das, was die Variable gerade nicht ist. Also wenn true mach false, wenn false mach true.

Wenn man nun in die Konsole geht und die Funktion toggleFlipFlop ausführt, ändert sich der Inhalt beim ersten Mal zu "ON", beim zweiten Mal wieder zu "OFF" und so weiter und so fort.
Das If-Statement
Ein if-Statement lässt das Programm unter einer bestimmten Bedingung (Condition) entscheiden, welcher Codeblock ausgeführt werden soll. Das Schlüsselwort else gibt den Block von Code an, der ausgeführt werden soll, wenn die Bedingung nicht erfüllt ist.
Funktionen
Eine Funktion ist ein Block aus Code, der auf Abruf ausgeführt wird. Er kann genauso mehrmals ausgeführt werden. Du kannst die Funktion rufen, indem du den Namen der Funktion in die Konsole eingibst der Entwicklertools, aber auch direkt in deinem Programm. Die Entwicklertools findest du mit "Rechtsklick -> Untersuchen" oder über die Tastenkombination "Strg Shift i".

Zu bestimmten Ereignissen, Events genannt kann man auch sagen, dass eine Funktion ausgeführt werden soll. Zum Beispiel beim Klick auf ein bestimmtes Element.
